# オセロゲーム

シンプルなブラウザベースのオセロゲームです。

## 使用方法

1. `index.html`をブラウザで開く
2. 黒（先攻）から開始
3. 有効な場所（薄緑色でハイライト）をクリックして石を配置
4. 相手の石を挟むと自動的に反転
5. 置ける場所がない場合は自動的にパス
6. 両者とも置けなくなったらゲーム終了

## 操作

- **クリック**: 石を配置
- **新しいゲーム**: ゲームをリセット

## ゲームルール

- 8×8のボードで対戦
- 相手の石を自分の石で挟むと反転
- 最終的に石の数が多い方が勝利
- 置ける場所がない場合は自動的にパス

## 技術仕様

- 単一HTMLファイル（外部依存なし）
- HTML5 + CSS3 + JavaScript (ES6)
- レスポンシブデザイン対応
